<h3>Setup & Run</h3>

---

<h4>**Run Locally**</h4>

1. Clone the repository
```
git clone https://github.com/aferrou/distributed_systems_assignment_2025_26.git
```

2. Build the project
```
mvn clean install
```

3. Start the application
```
mvn spring-boot:run
```

4. Access the application
```
http://localhost:8080
```
